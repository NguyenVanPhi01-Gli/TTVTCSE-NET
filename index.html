🌐 Hướng dẫn Publish lên Internet

Để nhiều người truy cập online, bạn cần đưa file HTML này lên một hosting. Dưới đây là các cách từ dễ đến khó:

🥇 GitHub Pages MIỄN PHÍ

Cách đơn giản nhất. Tạo tài khoản GitHub (miễn phí) → tạo Repository → upload file HTML → bật GitHub Pages → có link dạng ten-ban.github.io/ten-repo

1. Vào github.com → Đăng ký → New Repository
2. Upload file TTVTCHUSE_NET-v3.html → đổi tên thành index.html
3. Settings → Pages → Source: main branch → Save
4. Sau ~1 phút có link: username.github.io/repo-name

🥈 Netlify Drop MIỄN PHÍ

Nhanh nhất — không cần tài khoản. Kéo thả file HTML vào trang web Netlify.

1. Vào app.netlify.com/drop
2. Kéo thả file index.html (đổi tên trước) vào vùng drop
3. Ngay lập tức có link dạng: random-name.netlify.app
4. Đăng ký tài khoản để giữ link cố định

🥉 Google Drive (xem nội bộ) MIỄN PHÍ

Upload file lên Google Drive → Share link → đồng nghiệp tải về và mở bằng trình duyệt. Không xem trực tiếp online nhưng tiện chia sẻ nội bộ.

🏢 Server nội bộ (LAN) MIỄN PHÍ

Copy file vào máy tính/server nội bộ → cài IIS (Windows) hoặc Apache/Nginx (Linux) → truy cập qua IP nội bộ. Phù hợp dùng trong công ty không cần internet.

⚠️ Lưu ý bảo mật: Dữ liệu lưu trong localStorage trên từng máy. Nếu muốn nhiều người cùng sửa và đồng bộ realtime, cần kết nối Google Sheets hoặc nâng cấp lên backend (Firebase/Supabase). Liên hệ để được hỗ trợ thêm.
